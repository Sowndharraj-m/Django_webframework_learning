<h1>Virtual EnvironmentInstall Django -</h1>

Now, that we have created a virtual environment, we are ready to install Django.

Note: Remember to install Django while you are in the virtual environment!

Django is installed using pip, with this command:

(myworld) ... $ python -m pip install Django

Checking Django version -
To check if Django is installed correctly, you can run the following command:
**(myworld) ... $ python -m django --version**

(This will display the version of Django that is currently installed in your virtual environment. If you see the version number, it means Django has been successfully installed and is ready to use!)
